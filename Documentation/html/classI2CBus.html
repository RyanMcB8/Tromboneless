Detailed Description

Minimal Linux I2C bus wrapper for register-based communication.

This class:

  • Opens and closes the I2C device file (e.g. "/dev/i2c-1").
  • Provides methods to read/write blocks of data to/from 8-bit or 16-bit registers on I2C devices.

This class does not:

  • Handle any sensor-specific logic or protocols.
