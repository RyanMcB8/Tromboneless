A function that traces out the shape of an arc based upon the thickness of the arc and the gap between its inner sides. i.e the inner circle diameter that would remove an area from the larger outer circle.

Parameters
xThe x positon of the path, where this is the left most pixel of the area allowed for the path.
yThe y position of the path, where thes is the top most pixel of the area allowed by the path.
thicknessThe thickness of the arc itself.
gapThe inner diameter of the arc itself. The arcs center diameter would be the equivalent of gap + (thickness/2).
directionA boolean option where 0 represents the arc having its outer face facing right, and 1 represents the arc having its outer face facing left.
Return values
Returnsa juce::Path which traces the perimeter of an arc.

A function which returns a path matching the perimeter of a trombone.

Parameters
xThe x coordinate from which the paths minimum x starts. The default value for this is 300.
yThe y coordinate from which the paths minimum y starts. The default value for this is 300.
widthThe maximum width which the path is allowed to enclose. The default value for this is 50.
heightThe maximum height which the path is allowed to enclose. The default value for this is 20.
incPipesA boolean value determining if the trombones pipes should be included within the path or whether only the end of the pipe should be returned. The pipes are included by default.
Return values
Returnsa juce::Path which traces the perimeter of a trombone.

A function which can draw an arc with a specified thickness and nominal radius.

Parameters
xThe x position of the centre point of the arc. This is set to be 100 by default
yThe y position of the centre point of the arc. This is set to be 100 by default.
thicknessThe thickness of the arc which is being drawn. This is set to be 5 by default.
widthThe radius in the x axis of the arc. This is set to be 50 by default.
heightthe radius in the y axis of the arc. This is set to be 50 by default.
angularRangeThis sets the difference in minimum versus maximum angle of the arc which is being drawn. This is measured in radians and is set to be 5 by default.
rotationThis parameter rotates the arc to be facing the orientated the preferred way by the user. This function splits the angularRange in half from the up direction and has the left and right arcs of equal length. By default, this parameter is set to 0.
Note
This is built upon using the juce::centredArc function and line functions to create a path. If the user chooses to draw the shape outline as opposed to fill, this will be visible.
