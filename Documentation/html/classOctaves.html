A function which may be called upon to play a single note in a specified octave and reduce it from its peak to its normal amplitude.

Parameters
octaveThe octave in which the first harmonic is in.
noteThe note which is being played.
timeThe total amount of time the signal has been playing for.
tA parametric value which is used to determine how much the amplitudes of the waveforms should be reduced. When t=0, the waveforms are at their maximum amplitude and whent=0, their amplitudes = 0.
saturationThe normal relative amplitude of the signal to its peak.
Return values
Thisfunction will return a floating point value representing the normalised changed amplitude of the sigal at that point in time.
Note
The number of steps between 0 and 1 for the time value should be kept as high as possible as this represents the stage of oscillation the cosine wave is at.
