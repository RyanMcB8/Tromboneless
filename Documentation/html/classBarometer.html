A function which is used to change the background of the component to look like it is a barometer/ pressure gauge.

Parameters
gA pointer to the juce::Graphics instance which is being drawn to. This is likely the parent component or the main window of the system.
Note
This function may be altered to change the look of the device but it currently adds a white circle as a background, golden outer circle for the edge of it, 2 arcs around it corresponding to bounds, with lines connecting them representing the possible settings.
