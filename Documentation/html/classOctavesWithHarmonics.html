A function which may be called upon to stop playing a range of harmonic notes when a specific note is called upon to be played.

Parameters
nThe number of harmonics which should be applied to the signal.
octaveThe octave in which the first harmonic is in.
noteThe note which is being played.
timeThe amount of time the signal has been playing. This corresponds to the phase that the waveform is at.
tA parametric value which is used to determine how much the amplitudes of the waveforms should be reduced. When t=0, the waveforms are at their maximum amplitude and whent=0, their amplitudes = 0.
saturation
Return values
Thisfunction will return a floating point value representing the normalised changed amplitude of the sigal at that point in time.
Note
This function could be made to produce a much cleaner response by increasing the number of harmonics present and using non integer harmonics too.
