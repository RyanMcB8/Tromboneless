void _I2cFailRecover ( )
static

Expansion board i2c bus recovery

We may get reset in middle of an i2c access (h/w reset button, debug or f/w load) hence some agent on bus may be in middle of a transaction and can create issue or even prevent starting (SDA is low) this routine does use gpio to manipulate and recover i2c bus line in all cases.
