\doxysubsection{Detailed Description}
Represents a raw MIDI message as a sequence of bytes. 

This class is a lightweight value type used to store MIDI messages before transmission. It does not perform validation beyond basic structural construction.

A MIDI message consists of:\+
\begin{DoxyItemize}
\item A status byte
\item Zero, one, or two data bytes (for standard channel messages)
\item Or an arbitrary number of bytes (e.\+g. Sys\+Ex)
\end{DoxyItemize}

The underlying storage uses std::\+vector$<$unsigned char$>$ to allow variable-\/length messages for future extensibility.
