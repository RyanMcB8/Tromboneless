\doxysubsection{Detailed Description}
A class which is inherited from Juce:\+Component reponsible for setting the general style of a panel within the main window. 

\begin{DoxyNote}{Note}
A panel should represent at least one component such as an image, label, slider or another widget type. Similar types should be grouped together such as all calibration settings. 
\end{DoxyNote}
