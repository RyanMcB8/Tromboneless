\doxysubsection{Detailed Description}
A class which is inherited from the \doxylink{classCalibrationOnClick}{Calibration\+On\+Click} class that adds a dropdown menu next to the button for the user to select which option they wish to calibrate. 

\begin{DoxyNote}{Note}
This class was specifically designed for the embachure selction as no other widget needed to have a calibration button, hence the name of the class and lack of an intermediate step from \doxylink{classCalibrationOnClick}{Calibration\+On\+Click}. 
\end{DoxyNote}
