\doxysubsection{Detailed Description}
Time-\/of-\/\+Flight sensor interface using I2C and GPIO interrupt handling. 

This class manages a VL53\+L1X sensor. It communicates with the sensor over I2C, waits for a GPIO interrupt to indicate that a new measurement is ready, and publishes the measured distance via a callback.
