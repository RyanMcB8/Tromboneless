\doxysubsection{Detailed Description}
Minimal Linux I2C bus wrapper for register-\/based communication. 

This class\+:
\begin{DoxyItemize}
\item Opens and closes the I2C device file (e.\+g. "{}/dev/i2c-\/1"{}).
\item Provides methods to read/write blocks of data to/from 8-\/bit or 16-\/bit registers on I2C devices.
\end{DoxyItemize}

This class does not\+:
\begin{DoxyItemize}
\item Handle any sensor-\/specific logic or protocols. 
\end{DoxyItemize}
